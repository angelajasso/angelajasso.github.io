## 🧠 Sobre el proyecto

Este sitio web fue creado como mi portafolio personal, donde comparto mis proyectos y una muestra de mis habilidades actuales en desarrollo web.

Está diseñado con un enfoque en accesibilidad, modo oscuro/claro y estructura semántica, buscando mantener buenas prácticas desde el inicio de mi trayectoria profesional.
