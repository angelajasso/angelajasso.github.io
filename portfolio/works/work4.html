Juego Musical

GitHub Demo

Este proyecto es un juego interactivo musical desarrollado en JavaScript, que combina lógica de programación con creatividad para ofrecer una experiencia educativa.

El objetivo del juego es relacionar la música con la interacción digital. Los usuarios pueden tocar notas, activar sonidos y generar secuencias que les permiten jugar mientras aprenden principios básicos de ritmo y melodía.

La estructura del proyecto está construida con HTML y CSS, mientras que la interactividad se desarrolla completamente en JavaScript, utilizando el DOM y eventos para detectar clics y teclas. Además, aproveché la API de audio para generar sonidos en tiempo real.

El diseño es simple pero funcional, centrado en que la experiencia sea intuitiva: botones claros, colores diferenciados y retroalimentación visual inmediata para cada acción del jugador.

Más allá de la parte técnica, este proyecto refleja mi interés por unir la música y la programación. Desarrollarlo me permitió afianzar mis conocimientos en JavaScript, especialmente en manipulación de eventos y control del tiempo.

Este juego es un ejemplo de cómo la programación puede ser usada no solo para construir aplicaciones utilitarias, sino también experiencias creativas y divertidas.
